# 高光谱土壤有机碳（SOC）智能预测平台

基于田间高光谱反射率数据，用机器学习方法预测土壤有机碳含量，
为精准农业管理提供低成本的快速检测手段。
（本项目是区域碳汇计量研究方向的核心子模块之一——土壤碳库的快速预测；
后续研究计划进一步整合植被遥感监测与DNDC生物地球化学模型，
实现区域尺度的碳汇综合计量。）

## 背景

传统土壤有机碳检测依赖实验室化学分析，耗时耗力、成本较高。
本项目基于硕士阶段采集的田间高光谱反射率数据
（共35个采样点，覆盖400-1000nm波段），尝试用机器学习模型
替代部分实验室检测环节。

## 方法

- 数据预处理：异常值剔除、光谱去噪（SG平滑）
- 特征筛选：基于遗传算法的波段筛选
- 建模：PLSR / ANN / 1D-ResCNN 对比
- 评估指标：R²、RMSE，以及土壤建模领域常用的RPD
